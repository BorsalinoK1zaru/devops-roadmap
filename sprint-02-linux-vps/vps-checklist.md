# VPS Checklist

## Цель

Подготовить сервер для будущего деплоя Docker Compose проекта.

## Минимальные параметры VPS

- OS: Ubuntu 24.04 LTS
- CPU: 1 vCPU или больше
- RAM: 1–2 GB
- Disk: 20–25 GB SSD или больше
- Network: public IPv4
- Access: SSH key

## Что нужно получить после создания VPS

- IP-адрес сервера
- Имя пользователя
- Возможность подключиться по SSH
- Доступ через public key
- Открытый порт 22 для SSH

## Что нельзя коммитить

- Private SSH key
- Реальные пароли
- Токены
- `.env`
- `.env.dev`
- production `.env`
- приватные конфиги с настоящими IP/секретами

## Первые проверки на сервере

```bash
whoami
hostname
uname -a
cat /etc/os-release
id
groups
pwd
df -h
free -h
ss -tulpn
```

## Базовая логика диагностики сервера
1. Кто я на сервере?
2. Есть ли sudo-доступ?
3. Какая ОС?
4. Сколько свободного места?
5. Сколько RAM?
6. Какие процессы слушают порты?
7. Есть ли доступ в интернет?
8. Можно ли установить Docker?
